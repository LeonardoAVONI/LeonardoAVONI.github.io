ASWING Extended
User Manual

A comprehensive reference for the MIT ASWING aeroelastic solver — now live on the official website.

Context

ASWING is a program developed at MIT by Prof. Mark Drela for the aerodynamic, structural, and dynamic analysis of flexible aircraft. While powerful, its original documentation left significant gaps for new users. As part of my PhD work at ISAE-Supaero, I authored a substantially extended user manual to lower the barrier to entry for researchers and engineers.

What the Manual Covers

Theoretical Background

A concise but rigorous explanation of the beam-theory structural model, the vortex-lattice / lifting-line aerodynamic model, and their aeroelastic coupling within ASWING. Special attention is given to the assumptions and limitations that practitioners must understand.

Input File Structure

Detailed, annotated descriptions of every input-file keyword — geometry definition, structural properties (EI, GJ, mass distribution), aerodynamic parameters, engine/propulsion models, and trim conditions. Illustrated with worked examples.

Running Cases & Post-Processing

Step-by-step walkthroughs for static trim, dynamic linearisation (eigenvalue extraction), and time-domain simulation. Guidance on interpreting output files and visualising results with Python scripts.

Validation Cases

A set of benchmark cases — from a simple flexible beam to a representative HALE aircraft configuration — that allow users to verify their installation and gain confidence in the tool.

Impact

The manual is featured on the official ASWING website maintained by MIT, making it a reference resource for the international flexible-aircraft research community.
