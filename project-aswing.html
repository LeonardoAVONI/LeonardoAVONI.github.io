ASWING Extended
User Manual

A comprehensive reference for the MIT ASWING aeroelastic solver — now live on the official website.

Context

ASWING (MIT, Prof. Mark Drela) is the standard tool for flexible aircraft analysis. Its original documentation left significant gaps. As part of my PhD I authored an extended manual now hosted on the official MIT ASWING site.

Theoretical Background

Beam-theory structural model, vortex-lattice / lifting-line aerodynamics, and their aeroelastic coupling. Assumptions and limitations clearly stated.

Input File Reference

Annotated descriptions of every keyword: geometry, structural properties (EI, GJ, mass distribution), aero parameters, propulsion, trim conditions — with worked examples.

Running Cases & Post-Processing

Step-by-step walkthroughs for static trim, dynamic linearisation, and time-domain simulation. Python scripts for output parsing and visualisation.

Validation Cases

Benchmarks from a simple flexible beam to a representative HALE aircraft, allowing users to verify installation and build confidence.
