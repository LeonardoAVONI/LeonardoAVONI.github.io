Master Thesis · EPFL LIS Lab · 2022–2023

DroneDetective
Morphing Drone Wind Tunnel

Automated aerodynamic characterisation of a bio-inspired morphing drone — Master's thesis at EPFL's LIS lab.

Context

EPFL's LIS lab develops a morphing drone that changes wing shape mid-flight. Characterising its aerodynamic envelope requires sweeping a vast parameter space: airspeed, angle of attack, morphing configuration, yaw.

My Role

Design and implementation of DroneDetective — a fully automated wind-tunnel characterisation system.

Mechanical Setup

Force-balance mount with ATI 6-DOF force-torque sensor. Motorised angle-of-attack mechanism for automated incidence sweeps.

Automation Software

Python scripts orchestrating: set wind speed → command morphing → sweep incidence → acquire data → log. Runs overnight unsupervised.

Data Pipeline

Loads corrected for tare, inertia, and support interference. CL, CD, CM computed in polar form per configuration.

Results

Complete validated aero database across the full morphing configuration space — used by LIS for control design and performance optimisation.

Tech Stack

  • Python · ATI F/T sensor API · EPFL wind tunnel
  • SolidWorks · NumPy · Matplotlib
