Master Thesis Project · EPFL LIS Lab · 2022–2023

DroneDetective
Morphing Drone Wind Tunnel

Automated aerodynamic characterisation of a bio-inspired morphing drone — Master's thesis at EPFL's LIS lab.

Context

The LIS (Laboratory of Intelligent Systems) at EPFL develops bio-inspired robots, including a morphing drone capable of changing its wing shape mid-flight — inspired by birds. Understanding the aerodynamic envelope of such a vehicle requires characterising lift, drag, and pitching moment across a vast parameter space: airspeed, angle of attack, morphing configuration, and yaw angle.

My Role

I was responsible for designing and implementing a fully automated wind-tunnel characterisation system — DroneDetective — that could systematically explore this parameter space without manual intervention.

System Architecture

Mechanical Setup

Design and manufacturing of a force-balance mount that rigidly holds the drone in the test section while measuring 6-DOF loads (forces and moments) via a calibrated ATI force-torque sensor. A motorised angle-of-attack mechanism allows automated incidence sweeps.

Automation Software

Python control scripts orchestrate the entire test sequence: set wind-tunnel speed → command drone morphing configuration → sweep angle of attack → acquire sensor data → log results. The system runs overnight without supervision.

Data Pipeline

Raw force data corrected for tare loads, inertial effects, and support-structure interference. Aerodynamic coefficients (CL, CD, CM) computed and plotted in polar form for each configuration tested.

Results

A comprehensive aerodynamic database covering the morphing drone's full configuration space — providing the LIS lab with a validated dataset for control design and performance optimisation.

Tech Stack

Python · ATI F/T sensor API · EPFL wind tunnel instrumentation · SolidWorks · NumPy · Matplotlib
