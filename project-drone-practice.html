Personal Project · 2023–2024

First Drone Development
Practice

From CAD to firmware to first flight — building a quadcopter entirely from scratch.

Overview

With the theoretical design in hand, this phase turned equations into hardware — every component selected, assembled, wired, and programmed by hand.

1 · Frame Design & Manufacturing

250 mm configuration in FreeCAD / SolidWorks. Arms and body printed in PETG; carbon fibre arms planned for v2. Target: <250 g dry.

2 · Electronics Assembly

FC · 4-in-1 ESC · BLDC motors · LiPo · RC receiver · GPS. Soldering, wire routing, FC vibration isolation via damping mounts.

3 · Firmware & Configuration

ArduPilot / Betaflight flashing, sensor calibration, motor direction check, RC mapping, failsafe setup. PID tuning via Blackbox log analysis.

4 · SITL Simulation

Software-In-The-Loop testing in Gazebo to validate control gains before risking hardware.

5 · Field Testing & Iteration

First hover tests, stability assessment, iterative tuning. Documented crashes — a rite of passage.

Tools

  • FreeCAD / SolidWorks · FDM 3D Printing (PETG)
  • ArduPilot / Betaflight · Gazebo SITL
  • Python (log analysis)
