WingLoop
Automation & Control Library

Open-source Python library for flexible-aircraft simulation automation — presented at AIAA AVIATION FORUM 2025.

Motivation

Flexible-aircraft simulation workflows traditionally require manual setup at every step. WingLoop automates the entire pipeline and exposes a clean Python API for control design and parametric studies.

ASWING Wrapper & Automation

Scripted batch runs, automatic input-file generation, structured output parsing — eliminating manual interactive sessions.

Parametric Sweeps

Define any sweep (airspeed, altitude, payload, stiffness…) in a few lines. Results stored in pandas DataFrames.

Flutter Analysis

Automatic eigenvalue tracking across the flight envelope. Damping / frequency plots generated automatically. Flutter speed and divergence detection built-in.

Control Design Interface

Linearised state-space models exported from ASWING and interfaced with python-control / SciPy for LQR, H∞, and PID design.
