WingLoop
Automation & Control Library

Open-source Python library for flexible-aircraft simulation automation — presented at AIAA AVIATION FORUM 2025.

Motivation

Flexible-aircraft simulation workflows typically involve a tangle of manual steps: setting up trim conditions, sweeping parameters, extracting eigenvalues, and collecting results. WingLoop was born out of the need to automate this entire pipeline and expose a clean Python API for control design and parametric studies.

What WingLoop Does

ASWING Wrapper & Automation

WingLoop wraps the ASWING solver with a Python interface, enabling fully scripted batch runs, automatic input-file generation, and structured output parsing — eliminating the manual interactive session that ASWING traditionally requires.

Parametric Sweeps

Define any parameter sweep (airspeed, altitude, payload mass, structural stiffness…) in a few lines of Python. WingLoop handles the loop, monitors convergence, and stores results in structured pandas DataFrames.

Eigenvalue Tracking & Flutter Analysis

Automatic extraction and modal tracking across flight envelope sweeps. Damping and frequency plots generated automatically. Flutter speed and divergence detection built-in.

Control Design Interface

Linearised state-space models exported directly from ASWING and interfaced with Python control libraries (python-control, SciPy) for LQR, H∞, and PID design workflows.

AIAA AVIATION FORUM 2025

WingLoop was presented as a conference paper at AIAA AVIATION FORUM 2025 in Las Vegas, where it was received with interest from the flexible-aircraft and HALE UAV research community.
