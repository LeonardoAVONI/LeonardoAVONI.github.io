WingLoop
Automation & Control Library

Open-source Python library for flexible-aircraft simulation automation — presented at AIAA AVIATION FORUM 2025.

Motivation

Flexible-aircraft simulation workflows traditionally involve a tangle of manual steps: setting up trim conditions, sweeping parameters, extracting eigenvalues, and collecting results. WingLoop automates this entire pipeline and exposes a clean Python API for control design and parametric studies.

Key Features

ASWING Wrapper & Automation

Scripted batch runs, automatic input-file generation, and structured output parsing — eliminating the manual interactive session that ASWING traditionally requires.

Parametric Sweeps

Define any sweep (airspeed, altitude, payload, stiffness…) in a few lines. WingLoop handles the loop, monitors convergence, and stores results in pandas DataFrames.

Eigenvalue Tracking & Flutter Analysis

Automatic modal tracking across flight envelope sweeps. Damping / frequency plots generated automatically. Flutter speed and divergence detection built-in.

Control Design Interface

Linearised state-space models exported directly from ASWING and interfaced with python-control / SciPy for LQR, H∞, and PID workflows.

AIAA AVIATION FORUM 2025

WingLoop was presented as a conference paper at AIAA AVIATION FORUM 2025, Las Vegas, receiving interest from the HALE UAV and flexible-aircraft research community.
